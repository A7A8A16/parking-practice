# 项目记忆文件

## 项目信息
- **项目名称**: 极速驾考：地平线挑战 (Speed Drills: Horizon Challenge)
- **技术栈**: Three.js + Cannon-es 物理引擎 + Vite
- **启动命令**: `npm run dev` 或 `npx vite --host`（局域网访问）

## 用户偏好
- 使用中文进行交流

## 项目结构
- `src/` - 源代码目录
- `src/game/Game.js` - 游戏主逻辑（含触屏控制）
- `public/` - 静态资源
- `success.mp3` / `fail.mp3` - 音效文件
- `index.html` - 主页面及 UI 样式
